⟡ STXRZ.EXE PRIVACY POLICY

Effective Date: 2026

Overview

STXRZ.EXE is a gamified self-improvement system built for Discord. It uses XP, streaks, quests, and classes to track user progress and engagement.

Data We Collect

  • Discord User ID
  • XP and level progression
  • Streak and activity data
  • Selected class and preferences

How We Use Data

Data is used only for gameplay mechanics such as progression, leveling, quests, and tracking streaks.

Data Storage

All data is stored locally within the bot’s database system and is not sold or shared with third parties.

User Control

Users can reset or lose progress by leaving the system or server where the bot operates.

Security

We do not collect passwords, emails, or sensitive personal information. Only Discord-related identifiers are stored.

Purpose

This system is designed for productivity, education, and entertainment purposes only.

Contact

For issues or requests, contact the server administrator running STXRZ.EXE.
